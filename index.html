Markdown Parser in Haskell

Markdown parser implemented in Haskell using the Megaparsec library.

Instructions

To use the Markdown Parser, follow these simple steps:
  • Clone the Repository:
    • git clone https://github.com/your-username/markdown-parser.git
  • Build the Project:
    • cd markdown-parser cabal build
  • Run the Parser:
    • cabal exec markdown-parser inputFile.md outputFile.html Replace inputFile.md with the path to your markdown file and outputFile.html with the desired name for the HTML output.
